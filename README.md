# StayHub (пакет запуска)

Для запуска используйте только файл `StayHubStart.bat`. Все остальные служебные файлы находятся в каталоге `stayhub_package` и при обычной работе не требуются.

Если нужно открыть проект в IDE или запустить вручную:
1. Перейдите в папку `stayhub_package`.
2. Активируйте виртуальное окружение `venv` (если создано) или установите зависимости из `requirements.txt`.
3. Далее следуйте подробной инструкции в `stayhub_package/README.md`.
