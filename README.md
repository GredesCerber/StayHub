# StayHub

Учебный проект автоматизации работы мини-отеля: учёт номеров, гостей, броней, услуг и платежей. В корне расположен удобный лаунчер для демонстрации, а весь исходный код и зависимости собраны в каталоге `stayhub_package`.

## Быстрый старт (Windows)
- Запустите `StayHubLauncher.bat` через двойной клик или из PowerShell (`./StayHubLauncher.bat`).
- В меню доступны: запуск сервера, заполнение базы тестовыми данными, открытие панели администратора.
- Для полноценной демонстрации сначала выберите пункт заполнения базы, затем — запуск сервера (браузер откроется автоматически).

> Если PowerShell блокирует запуск, выполните в текущем окне команду `Set-ExecutionPolicy -Scope Process RemoteSigned`, затем повторите запуск лаунчера.

## Когда нужен доступ к коду
- Перейдите в каталог `stayhub_package`.
- Настройте виртуальное окружение и зависимости по инструкции из `stayhub_package/README.md`.
- Там же описаны структура проекта, REST API и сценарий демонстрации.

## Состав репозитория
- `StayHubLauncher.bat` — основной запускной файл с меню.
- `stayhub_package/` — исходный код приложения, база данных, виртуальное окружение (если создано).
- `stayhub_package/README.md` — подробная документация, технические детали и инструкции для разработчиков.
